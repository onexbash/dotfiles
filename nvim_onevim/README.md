# onexbash - onevim

## Requirements
- neovim >= 0.12
- git >= 2.19
- tar
- curl
- tree-sitter-cli
- C compiler in $PATH
- A [Nerd Font](https://www.nerdfonts.com/) installed & set for your Terminal Emulator
- [luarocks](https://luarocks.org/)

## Plugins
### lazy.nvim
The Plugin Manager we are using to handle Plugin Configurations & Installations
